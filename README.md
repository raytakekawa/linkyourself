# README

## アプリケーションの概要
このアプリでできることは、シンプルです。
例えば、あなたがイメージするものがあるとしましょう。
多くの場合ぼやっとした絵のようなイメージなのではないでしょうか。
これまでのアプリケーションでは、カテゴリーや単語入力などの操作を介してイメージを具体化して検索します。
しかし、今回私が開発したアプリケーションではそういった方法ではなく直感的な検索を実現します。
